About Noendrise

Noendrise is a free real-time online stair climbing game built for modern desktop and mobile browsers.

What Makes It Different

Noendrise turns a simple climbing challenge into a live competition. Players climb floor by floor, use items, hide from attacks, receive system notices, and compete on player and country leaderboards.

Global Competition

Every run contributes to a live ranking experience. Players can represent their country, chase higher floors, earn titles, and compare scores with climbers around the world.

Browser First

The game runs directly in the browser with no installation required. It supports keyboard controls on desktop and touch controls on mobile.
